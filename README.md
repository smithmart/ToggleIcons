# ToggleIcons

Toggles Windows desktop icon visibility. Run once to hide, run again to show.

## Requirements

- [.NET 10 SDK](https://dotnet.microsoft.com/download)
- Windows

## Build

```
dotnet build src/toggle-icons.csproj
```

## Run

```
dotnet run --project src/toggle-icons.csproj
```

Or run the compiled binary directly:

```
src/bin/Debug/net10.0-windows/toggle-icons.exe
```
